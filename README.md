### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/iion91/covid.py.git
   ```
   or download the file
2. Install dependencies
   ```sh
   pip install numpy
   pip install matplotlib
   ```
3. Start the script 
   ```cmd
   python virus.py
   ```
